# RPG-JVJ
Trabalho de E.D.1 sobre um jogo de RPG com algumas condições

# Planejamento do Trabalho

O trabalho será dividido em 3 partes e cada uma delas será explicada da seguinte forma:

## 1. Implementação do TAD "Personagem"

O TAD "Personagem" será estruturado nos seguintes arquivos:
* .h
    * Criar o TAD "Personagem", que precisará ter os seguintes elementos:
        * Nome
        * Nível (1 -> 20)
        * Dado (4, 6, 8, 10, 12, 20) que determina a rolagem através de um número aleatório.
        * Iniciativa (nível + rolagem do dado)

    * Definir o cabeçalho das funções do .c.
* .c
    * Definir as seguintes para o TAD:
        * Criar personagem (Nome, dado, nível): A questão da iniciativa é definida pelo nível + rolagem. A rolagem é obtida quando um número aletório entre 1 e o "dado" é escolhido.

        OBS: Criar personagens através da leitura de um arquivo .txt.

        * Rolagem (*Personagem): Define a rolagem com base no "dado" escolhendo um número aleatório.
